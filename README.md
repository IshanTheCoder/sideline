# Anchor

Anchor is a sideline companion for coaches to capture in-game observations through quick voice notes and lightweight sketches, preserving insights for post-game review without disrupting focus.

## Project Status
This project is in the early exploration and design phase. Current work focuses on understanding coaching workflows and validating the problem before building a full implementation.

## Motivation
Coaches often notice important patterns and issues during live play that are difficult to capture and easy to forget by the time film review begins. Anchor is designed to quietly support memory, not replace coaching judgment or strategy.

## Non-Goals
- This is not a strategy or analytics engine
- This does not generate coaching decisions
- This does not replace film or existing tools

## Next Steps
